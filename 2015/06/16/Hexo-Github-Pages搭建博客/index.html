流程

  1. 开通githubpage
  2. 本地安装git、node.js、hexo
  3. 添加域名解析，指向githubpage

开通githubpage

在github上创建一个新的repo，命名为username<github的username>.github.io

进入repo，点击setting，找到Github Pages，将source改为master，并点击save保存

安装git、node、hexo
